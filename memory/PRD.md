# CropVision AI — PRD

## Original problem
Satellite-based crop disease detection & advisory platform for smallholder farmers, agronomists, and cooperatives. Production-grade, multilingual, RBAC-secured, with async inference pipeline, LangGraph reasoning, and n8n orchestration.

## Stack (adapted to platform)
- Backend: FastAPI + MongoDB (2dsphere geo index) + in-process async queue (drop-in for BullMQ)
- AI: Claude Sonnet 4.5 via emergentintegrations (vision + LangGraph reasoning)
- Frontend: React 19 + Tailwind + framer-motion + recharts + sonner + react-i18next (EN/HI)
- Orchestration: n8n workflow JSON exported + native FastAPI mirror

## User personas
- **Farmer** — sees own fields, receives alerts in own language (EN/HI), can trigger scans
- **Agronomist** — reviews escalated critical cases from a shared queue
- **Admin** — monitors pipeline health, queue stats, audit trail

## Implemented (2026-02)
- JWT auth (access+refresh rotation), RBAC middleware, rate limiting, audit log
- Fields CRUD with 2dsphere geo index + owner-scoped visibility
- Async inference pipeline: enqueue → Claude vision → 5-node LangGraph (diagnose_refine → severity_score → advisory_localize → confidence_gate → escalation_decide) → detection + alert persist, retries + dead-letter
- Bilingual advisories (EN + Hindi) generated by LangGraph
- Cinematic landing page (spectral green theme, art-directed)
- Dashboard with real-time fleet metrics, health trend chart, field cards, alert feed
- FieldDetail with 14-day trend + reasoning trace viewer
- Agronomist queue + Admin pipeline/audit views
- Observability endpoints: /api/health, /api/metrics, /api/admin/pipeline
- n8n workflow JSON exported with env placeholders

## Iteration 2 (2026-02) — P1 backlog delivered
- **Twilio WhatsApp**: `twilio_delivery.py` module wired into inference pipeline. Auto-detects env vars — mock mode when unset (persists whatsapp alert row with `delivered=False` + `delivery_error`; pipeline never fails). Set `TWILIO_ACCOUNT_SID`, `TWILIO_AUTH_TOKEN`, `TWILIO_WHATSAPP_FROM` in `backend/.env` to go live.
- **Multi-tenant cooperatives**: `Cooperative` model + collection, `cooperative_id` on User & Field, new `coop_admin` role. Endpoints: `GET /api/cooperatives` (public list), `POST /api/cooperatives` (admin), `GET /api/coop/dashboard` (coop_admin). RBAC scoping added to `list_fields`, `list_detections`, `list_alerts` for coop_admin.
- **Field polygons**: `GeoPolygon` model, `POST /api/fields` accepts polygon array `[[lng,lat],...]`, auto-closes ring, persists as GeoJSON.
- **Real map**: `react-leaflet` + OpenStreetMap tiles (no token needed) with custom dark-tile filter matching the spectral theme. Leaflet-draw polygon tool integrated into Add-Field modal. Farmer dashboard + Field detail + Coop dashboard all show maps.
